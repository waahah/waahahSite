# *waahahSite*
*Git 提交后自动部署我的个人博客到 Vercel 并添加netlifyCms为后台编辑组件*

在线预览地址：[无名小站](https://www.waahah.xyz/)

### Built with
```cmd
git clone https://github.com/waahah/waahahSite.git
cd waahahSite
npm install
hexo clean
hexo generate
hexo server
```
