### 前言

git如何更新以及git安装包

### 一、git如何更新？

步骤：  
1.先查看自己的git版本  
`PowerShell`命令输入：

```bash
git --version

```

2.更新git  
当前git版本是2.17.1之前的，输入：

```bash
git update

```

当前git版本是2.17.1之后的，输入：

```bash
git update-git-for-windows

```

### 二、git安装包下载途径

git官方下载：[速度慢](https://git-scm.com/download/win)
