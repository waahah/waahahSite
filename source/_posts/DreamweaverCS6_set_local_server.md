## 前言
<p>使用 ASP、 ColdFusion 或 PHP 等服务器端语言开发网站时， 可通过本地测试服务器
查看动态代码生成的页面的实际效果， 然后将它转入生产阶段。 如果您要使用动态页面进行
任何预览， Dreamweaver 实际会要求您设置测试服务器。</p>
